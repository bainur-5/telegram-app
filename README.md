# Telegram App (Static HTML)

Это простая статическая верстка: HTML + SCSS/CSS + Vanilla JS.
Откройте `index.html` двойным кликом в браузере или через любой live-server из корня.
Остальные страницы находятся в папке `pages/`.
Подключение на страницах: `assets/css/style.css` и `assets/js/main.js`.
Бинарные шрифты не используются — оставлен системный стек шрифтов.
(Опционально) пересборка стилей: `npx sass assets/scss/style.scss assets/css/style.css --no-source-map`.
