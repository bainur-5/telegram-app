# Telegram App (Static HTML)

Простая статическая верстка: HTML + SCSS/CSS + Vanilla JS.
Откройте `index.html` двойным кликом в браузере или через live-server.
Внутренние страницы лежат в `page/`.
Стили подключаются из `style/style.css`, скрипты — из `script/main.js`.
SVG-иконки находятся в `style/img/`.
(Опционально) пересобрать CSS: `npx sass style/style.scss style/style.css --no-source-map`.
Бинарные файлы не используются.
