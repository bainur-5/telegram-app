# HTML/CSS Deliverable

Откройте `index.html` или любой файл из `pages/*.html` напрямую в браузере.

## Структура
- `css/` — reset, variables, ui, pages
- `js/` — минимальные интерактивы (tabs/dropdown/modal/switch)
- `assets/` — иконки из исходного проекта
- `assets/fonts/` — папка под шрифты (TTF добавляет клиент вручную)
