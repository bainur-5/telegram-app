# Mapping React -> HTML/SCSS

- `TopBar.tsx` -> `deliverable/pages/*.html` `.topBar*`; styles -> `deliverable/scss/components/_top-bar.scss`.
- `Navigation.tsx` + `CutCornerButton.tsx` -> `deliverable/pages/*.html` nav; styles -> `deliverable/scss/components/_navigation.scss` + `deliverable/scss/ui/_buttons.scss`.
- `MiningPage.tsx` -> `deliverable/pages/mining.html`; styles -> `deliverable/scss/pages/_mining.scss` + `deliverable/scss/ui/_cards.scss`.
- `ReferralsPage.tsx` -> `deliverable/pages/referrals.html`; styles -> `deliverable/scss/pages/_referrals.scss` + `deliverable/scss/components/_friends-card.scss`, `_card-text.scss`, `_referral-history.scss`.
- `MarketPage.tsx` + `CreateOrderModal.tsx` -> `deliverable/pages/market.html`; styles -> `deliverable/scss/pages/_market.scss`, `_market-controls.scss`, `_market-item-card.scss`, `_modal.scss`.
- `ProfileSettingsPage.tsx` -> `deliverable/pages/profile-settings.html`; styles -> `deliverable/scss/pages/_profile-settings.scss`, `_history-section.scss`, `_notification-card.scss`, `_language-card.scss`, `ui/_switch.scss`.
- `FinancesPage.tsx` -> `deliverable/pages/finances.html`; styles -> `deliverable/scss/pages/_finances.scss`, `_finance-tabs.scss`, `_deposit-panel.scss`, `_withdraw-panel.scss`, `_glass-card.scss`.
